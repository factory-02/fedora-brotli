# Информация

SPEC-файл для создания RPM-пакета **brotli**.

## Установка

1. Подключить репозиторий **METASTORE**: `dnf copr enable metastore/fedora-brotli`.
2. Установить пакет: `dnf install brotli`.

## Авторы
